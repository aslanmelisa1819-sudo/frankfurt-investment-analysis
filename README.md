Frankfurt Autonomous Retail Investment Analysis
Data-driven location intelligence system for autonomous retail investment decisions in Frankfurt. Identifies optimal districts by analyzing demographics, foot traffic, and competitor density.
Data Sources: Destatis (Federal Statistical Office) & Offene Daten Frankfurt
Tools: Python, SQL (Google BigQuery), Streamlit, Plotly
Output: Interactive map highlighting top 3 investment locations with €1.4M investment model and annual profit estimates.
